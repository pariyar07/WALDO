    Introduction

    Welcome! Get started with WALDO and build web pages in no time. We provide the pre made components to speed up your process. You just need to add our stylesheet to you HTML page and copy-paste the HTML code given and customize your content.

    Quick Start

    To use this framework, you simply need to put the below link in your header file above all other style sheets.

    /* html */
    <link rel ="stylesheet" href = "https://waldo-ui.netlify.app/styles/main.css" />     /* CSS */
    @import url("https://waldo-ui.netlify.app/styles/main.css")

    Customize

    Athough WALDO tries to implement its best but sometimes user wants their own styles dealing. For that you can simply add your own class to the existing classes.

    Hop On

    If you have copied the above link and pasted inside the head tag. You are now ready to make your website come to life, click on the button below to get started.
